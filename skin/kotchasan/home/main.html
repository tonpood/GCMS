คชสารเว็บเฟรมเวิร์ค

ช้างนอกจากจะเป็นสัญลักษณ์ของ PHP แล้ว ยังเป็นสัญลักษณ์ประจำชาติของเราอีก ผมเลยเลือกที่จะใช้ชื่อนี้เป็นชื่อของ Framework ที่ออกแบบโดยคนไทย 100%

แนวคิด

ผมออกแบบ คชสาร โดยมีแนวคิดเริ่มต้นที่จะใช้เป็นแกนในการพัฒนา GCMS ดังนั้นนิยามแรกของผมเลยก็คือ "เร็ว" ถัดมาก็เพื่อให้สามารถใช้งานเฟรมเวิร์คได้ง่ายที่สุด เนื่องจากเฟรมเวิร์คที่มีในปัจจุบัน บางอันก็ต้องการ Server ที่พิเศษ บางอันก็มีการติดตั้งที่ยุ่งยาก และสุดท้ายเลย ก็เพื่อให้เกิดความตื่นตัวว่าคนไทยเราก็สามารถพัฒนาเฟรมเวิร์คใช้กันเองได้

คุณสมบัติ

รวดเร็ว

ทำงานได้รวดเร็ว ใช้หน่วยความจำเริ่มต้นน้อย ช่วยให้เว็บไซต์เปิดได้เร็วขึ้น มีผลให้สามารถรองรับจำนวนผู้เยี่ยมชมได้มากขึ้น

ครบครัน

ประกอบด้วย PHP Framework, Javascript Libraly และ CSS Framework ที่ออกแบบให้ทำงานร่วมกัน

Database

ทำงานร่วมกับฐานข้อมูลอย่างง่ายๆ ด้วย Recordset, QueryBuilder และ CRUD (เบื้องต้นสนับสนุนเฉพาะ MySQL)

หลายภาษา

สนับสนุนการสร้างระบบเว็บไซต์หลายภาษา มีไลบราลี่สำหรับจัดการภาษาทั้งในส่วนของ PHP และ Javascript

สถาปัตยกรรม MMVC

Modules Model View Controller โมดูลของ MVC หลายๆโมดูลช่วยให้การขยายระบบ ทำได้ง่ายๆ

Template Engine

แสดงผลเว็บไซต์บนหลากหลายอุปกรณ์ (Responsive) และแสดงผลเหมือนๆกันในทุกบราวเซอร์

โมดูลหลากหลาย

Data Table, Ajax Form, Ajax Upload, Form Javascript, Javascript Validator, Responsive Menu

PSR

สนับสนุนการใช้งานร่วมกับ Libraly อื่นๆที่ออกแบบตามมาตรฐาน PSR-1, PSR-2, PSR-3, PSR-4, PSR-6, PSR-7

ความต้องการ

  • PHP 5.3 ขึ้นไป (สนับสนุน PHP7 แล้ว)
  • PDO MySQL สำหรับการใช้งานร่วมกับ Database
  • Multibyte Extension สำหรับการสนับสนุน UTF-8
